# Lahore Wander Lust

Lahore Wander Lust is a comprehensive travel and tourism application designed to showcase the rich cultural heritage, historical landmarks, and vibrant lifestyle of Lahore, Pakistan. This application serves as a digital guide for tourists and locals alike, providing detailed information about attractions, food spots, events, and hidden gems across the city.

## Features

- **Explore Lahore**: Discover historical sites, cultural landmarks, and modern attractions.
- **Food Guide**: Explore the best culinary experiences Lahore has to offer, from traditional cuisine to modern cafes.
- **Events & Festivals**: Stay updated with the latest events, festivals, and cultural activities happening in the city.
- **Travel Planner**: Plan your trip with curated itineraries and recommendations.
- **User Reviews & Ratings**: Share your experiences and read reviews from other travelers.
- **Interactive Map**: Explore Lahore with an integrated map showing all points of interest.

## Tech Stack

- **Frontend**: React, Vite, TypeScript
- **Styling**: Tailwind CSS, Shadcn UI
- **State Management**: Zustand
- **Routing**: React Router DOM
- **Mapping**: Leaflet
- **UI Components**: Radix UI

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm (or yarn/pnpm)

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd lahore-wanderlust-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:

```bash
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) to view it in your browser.

### Build

Build the application for production:

```bash
npm run build
```

### Run Tests

Run the test suite:

```bash
npm test
```
